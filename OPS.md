## Operational Notes

- Add durable repo-specific lessons here only when they change future agent behavior.
- AC-2 outcomes are clusterer-recall-dependent and monotone-strict: the corroboration rule (`evaluate_ac2`) fails a trial on any BR-2+ finding with `agreement_count >= 2`, and agreement is only counted when the lexical clusterer merges cross-slot paraphrases. Improving clustering recall can only make AC-2 stricter (never quietly green). Any change to `aggregate.py` clustering/threshold MUST re-run AC-2 before claiming green. See `docs/design-note-ac2-corroboration-rule.md`.
- Fixture/corpus content pins are SHA-256 over newline-normalized text (`Path.read_text` normalizes line endings). Do not compute pins from raw bytes on a CRLF working copy — the values will not match.
